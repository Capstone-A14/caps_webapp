# About the Code

This is a piece of code for the client-side or front-end aspect of our Capstone Project: Cardimetry! This code was crafted with React, Next.JS, TailwindCSS, and D3.JS to enhance the look and feel for users utilizing our web application. The user interface design can seamlessly adapt to screens of different sizes. For more details about our project, see https://github.com/Capstone-A14.
